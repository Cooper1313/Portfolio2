OutfIQ - Smart Closet System

IoT-powered outfit recommendations based on real-time weather data

Python IoT APIs JavaScript Arduino 3D Modeling

Project Overview

OutfIQ is a smart closet system that integrates Python, APIs, and IoT technologies to recommend outfits based on live local weather data. The system pulls real-time data from a nearby weather station and matches it with items from a virtual wardrobe, aiming to create a seamless blend of climate awareness and intelligent fashion.
